# Chain Reaction React

This folder contains a React + TypeScript reimplementation of the original Chain Reaction Java desktop game, plus a WebSocket server for online multiplayer rooms (up to four players).

## Getting Started

```bash
npm install
```

### Start the WebSocket Server

```bash
npm run server
```

The WebSocket server listens on `ws://localhost:8080` by default.

### Start the Vite Dev Server

```bash
npm run dev
```

The dev server runs on `http://localhost:4173` by default.
